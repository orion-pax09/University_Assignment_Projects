# University Assignment Projects (BS Software Engineering)

This repository contains my university assignments, lab work, and academic projects completed during my **Bachelor's degree in Software Engineering**.

The repository is organized by **semester and subject** to keep the work structured and easy to navigate.

## Purpose of this Repository

* Organize my academic work in one place
* Track my learning progress throughout the degree
* Practice version control using Git and GitHub
* Maintain a portfolio of my programming projects

Each folder represents a **semester**, and inside it are the **subjects and assignments** completed during that time.
